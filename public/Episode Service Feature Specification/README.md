# Episode Service Feature Specification

이 프로젝트는 Figma Make를 통해 생성된 목업 코드를 실제 작동하는 프로토타입으로 변환한 것입니다.

## 주요 기능

### 1. 사용자 인증 및 온보딩
- 랜딩 페이지
- 소셜 로그인 (카카오, Google - 목업)
- 경험 뱃지 선택

### 2. 마인드맵 캔버스
- 노드 생성, 수정, 삭제
- 드래그 앤 드롭으로 노드 위치 이동
- Space + 드래그로 캔버스 이동
- Alt + 스크롤로 확대/축소
- 노드 공유 기능
- 탭 기반 멀티 뷰

### 3. AI 챗봇
- STAR 방식 경험 정리 지원
- 실시간 대화형 인터페이스
- 진행 상황 시각화 (S-T-A-R)
- 자연스러운 응답 시뮬레이션

### 4. 공백 진단
- 기업별/직무별 문항 기반 진단
- 답변 어려운 역량 추출
- 추천 인벤토리로 표시

### 5. STAR 에디터
- AI가 생성한 초안 편집
- STAR 구성 요소별 정리
- 저장 및 복사 기능

### 6. 대시보드
- 저장된 STAR 에셋 목록
- 검색 및 필터링
- 상세 보기 및 복사/삭제

### 7. 공유 기능
- 노드별 공유 링크 생성
- URL 기반 공유 뷰
- 실시간 동기화 (localStorage 기반)

## 기술 스택

- **React 18.3** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Vite** - 빌드 도구
- **React Router 6** - 라우팅
- **Tailwind CSS** - 스타일링
- **Radix UI** - UI 컴포넌트
- **Lucide React** - 아이콘

## 설치 및 실행

### 의존성 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

개발 서버는 기본적으로 `http://localhost:3000`에서 실행됩니다.

### 빌드
```bash
npm run build
```

## 프로토타입 특징

### 데이터 저장
- 모든 데이터는 **localStorage**에 저장됩니다
- 브라우저를 닫아도 데이터가 유지됩니다
- 다른 탭과 실시간 동기화됩니다

### 목업 데이터
- 소셜 로그인은 실제 인증 없이 목업으로 작동합니다
- AI 챗봇은 키워드 기반 응답을 생성합니다
- 공백 진단은 미리 정의된 문항을 사용합니다

## 주요 개선 사항

1. **React Router 도입**: URL 기반 라우팅으로 공유 링크가 제대로 작동
2. **AI 챗봇 개선**: 더 자연스러운 응답과 타이핑 인디케이터 추가
3. **에러 처리**: try-catch 블록과 사용자 친화적 에러 메시지
4. **로딩 상태**: 비동기 작업 시 시각적 피드백
5. **데이터 검증**: localStorage 데이터 형식 검증
6. **UX 개선**: 애니메이션, 피드백 메시지 등

## 사용 방법

1. **시작하기**
   - 랜딩 페이지에서 "무료로 시작하기" 클릭
   - 소셜 로그인 선택 (목업)
   - 경험 뱃지 선택

2. **마인드맵 사용**
   - 노드 클릭: 선택
   - 노드 더블클릭: 수정
   - 노드 드래그: 위치 이동
   - 노드에 마우스 오버: 하위 노드 추가 버튼 표시
   - Space + 드래그: 캔버스 이동
   - Alt + 스크롤: 확대/축소

3. **AI 챗봇 사용**
   - 노드 선택 후 AI 챗봇에서 대화 시작
   - STAR 방식으로 경험을 설명
   - 모든 요소(S-T-A-R) 완성 시 초안 생성 가능

4. **공유하기**
   - 노드 우클릭 → "공유하기"
   - 생성된 링크를 복사하여 공유

5. **대시보드**
   - 저장된 STAR 에셋 확인
   - 검색 및 필터링
   - 복사 및 삭제

## 주의사항

- 이 프로토타입은 **완전한 프로토타입**입니다
- 실제 백엔드 서버가 없으며 모든 데이터는 localStorage에 저장됩니다
- AI 기능은 시뮬레이션입니다
- 소셜 로그인은 목업입니다

## 라이선스

이 프로젝트는 Figma Make를 통해 생성된 코드를 기반으로 합니다.
