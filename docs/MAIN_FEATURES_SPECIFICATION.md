# 📖 주요 기능 리스트

## ❶ 기출문항 셀프진단

**`[한줄설명]`**

희망 직무별 기업의 최근 **기출 자기소개서 문항**을 통해 추가 정리가 **필요한 역량 범주**를 분석한다.

**`[세부설명]`**

1. 기출문항 셀프진단은 별도 진단 페이지 또는 마인드맵 캔버스 내에서 실행이 가능하다.
2. 공채모집이 진행되는 기업들의 최근 자기소개서 기출 문항을 제공한다.
3. 유저는 개별 질문을 검토하며 해당 질문에 답변할 활동/에피소드가 있는지 체크하고 이를 기반으로 유저가 추가로 정리하면 좋을 역량 범주를 제공한다.

**`[세부기능]`**

> **진단과정**
> 
- 유저가 온보딩 과정에서 선택한 희망직무를 기출문항 셀프진단탭 상단에 노출시킨다.
- 유저가 선택한 직무를 모집한 전적이 있는 기업들의 목록과 해당 기업의 기출 자기소개서 문항을 제공한다.
- 유저가 각 자기소개서 문항에 '가능함/모르겠음'으로 답변하면 '모르겠음'을 답변한 질문들을 기반으로 유저가 보충하면 좋을 '역량 빈틈' 키워드를 제공한다.

> **진단 결과 조회**
> 
- 진단 결과 도출된 '역량 빈틈' 키워드를 마인드맵 사이드바에서 조회할 수 있다.
- '역량 빈틈' 리스트를 마인드맵 캔버스에 상시적으로 보여준다.
- 별도 페이지에서 진행하는 경우
    - 진단 완료 후 '경험 정리하러 가기' 버튼을 통해 마인드맵 생성 플로우로 바로 이동할 수 있다.
    - 진단 결과는 자동으로 저장되어 마인드맵 생성 시 활용된다.

## ❷ 마인드맵 기반 활동정리

**`[한줄설명]`**

취업 준비과정에 활용할 활동내용을 **마인드맵**을 통해 체계적으로 정리한다.

**`[세부설명]`**

1. 계층형으로 구조화된 마인드맵
    - **[주제(중심노드) - 경험 카테고리 - 경험 - 에피소드]** 계층이 노드 추가에 따라 자동으로 적용된다.
2. 마인드맵 타입 설정
    - 개인 마인드맵 : 본인 개인의 경험을 정리하기 위해 활용
    - 팀 마인드맵 : 여러 사용자가 동시에 편집할 수 있으며 공유 기능이 활성화됨
3. 실시간 협업 기능
    - 팀 마인드맵에서 여러 사용자가 동시에 편집할 수 있음
    - 현재 편집 중인 사용자 목록을 실시간으로 표시
    - 로그인 여부에 따라 읽기 전용 모드와 편집 모드 구분

**`[세부기능]`**

> **마인드맵 생성**
> 
- 개인 마인드맵/팀 마인드맵 중 선택
- 개인 마인드맵: 배지(경험 카테고리) 선택 후 프로젝트 생성
- 팀 마인드맵: 경험 노드 직접 입력 후 프로젝트 생성
- 프로젝트 타입은 생성 후 변경 불가

---

> **팀 마인드맵**
> 
- '공유' 기능을 통해 단일 마인드맵을 여러 사람이 동시에 편집 가능
- 공유 링크 생성 및 관리
- 실시간 편집자 표시 (현재 편집 중인 사용자 목록)
- 읽기 전용 모드 지원 (비로그인 사용자)

---

> **마인드맵 캔버스 조작**
> 
- 노드 생성
    - 노드 우클릭 메뉴에서 하위 노드 추가
    - 노드 연결점(상하좌우) 클릭으로 하위 노드 추가
    - 툴바의 '노드 추가' 버튼으로 캔버스 빈 공간에 자유 노드 생성
    - 캔버스 빈 공간 우클릭으로 노드 생성
    - 자유 노드는 드래그하여 기존 노드와 연결 가능
- 노드 편집
    - 노드 더블클릭으로 내용 인라인 편집
    - 노드 우클릭 메뉴에서 편집 옵션
    - 노드 삭제 (연결된 하위 노드까지 함께 삭제)
- 캔버스 네비게이션
    - 커서 모드: '선택' 모드와 '이동' 모드 전환
        - 선택 모드: 노드 선택 및 편집 가능
        - 이동 모드: 캔버스 팬(pan) 가능
    - 트랙패드/마우스 휠로 줌 인/아웃 (커서 위치 중심)
    - '화면 맞춤' 버튼으로 모든 노드를 화면에 맞춤
- 레이아웃 설정
    - 원형 레이아웃: 중심 노드를 기준으로 원형 배치
    - 트리형 레이아웃: 좌우 대칭 트리 구조 배치
    - 브랜치 레이아웃: 중심에서 좌우로 뻗어나가는 직각 브랜치 배치
    - 레이아웃 변경 시 자동 재배치
- 시각화 설정
    - 그리드 표시/숨기기
    - 색상 테마 선택 (기본, 파스텔, 비비드, 모노크롬)
    - 브랜치별 색상 자동 할당 (경험 노드 기준)

---

> **검색 및 필터**
> 
- 키워드 검색: 노드 내용에 포함된 키워드 검색(검색된 노드 하이라이트 표시)
- 역량 필터 : 에피소드의 STAR 태그 기반 역량 필터링(필터링된 노드 하이라이트 표시)

---

> **내보내기**
> 
- 이미지로 저장 (PNG)
- PDF로 저장

---

> **탭 관리**
> 
- 여러 마인드맵 프로젝트를 동시에 탭으로 열기
- 새 탭 추가 버튼으로 프로젝트 열기 또는 새 프로젝트 생성
- 탭 닫기 기능
- 새로고침 후에도 열려있던 탭 자동 복원

---

> **사이드바**
> 
- STAR 정리하기
    - 마인드맵 캔버스에서 에피소드 노드 선택 시 사이드바 STAR 에디터 열기
    - Situation, Task, Action, Result 각 항목별 입력
    - 강점/역량 태그 추가 및 관리
    - 작성 완료된 STAR 내용 저장 및 조회
- 기출문항 셀프진단(주요 기능 ❶)

## ❸ 에피소드 보관함

**`[한줄설명]`**

마인드맵에서 정리한 모든 에피소드를 STAR 형식으로 한눈에 조회하고 편집할 수 있는 통합 관리 페이지

**`[세부설명]`**

1. 마인드맵별 에피소드 목록 조회
    - 개인 마인드맵과 팀 마인드맵을 탭으로 구분하여 조회
    - 각 프로젝트별로 에피소드를 그룹화하여 표시
    - 마인드맵별 테이블 뷰로 구조화된 정보 제공
    - 각 에피소드의 Situation, Task, Action, Result를 테이블 칼럼으로 표시
2. 내용 편집
    - 에피소드별 STAR 내용 편집
    - 역량 태그 추가 및 수정
    - 경험 기간 정보 추가 및 수정

**`[세부기능]`**

> **조회 기능**
> 
- 프로젝트별 그룹화 : 모든 에피소드를 프로젝트별로 그룹화하여 표시
- 탭 필터 : 전체 / 개인 마인드맵 / 팀 마인드맵 탭으로 필터
- 프로젝트 필터 : 드롭다운으로 특정 프로젝트만 조회
- 카테고리 필터 : 배지 타입(인턴, 동아리, 프로젝트 등)별 필터링
- 역량 태그 필터 : 에피소드에 태그된 역량 키워드로 필터링
- 검색 : 에피소드명, 경험명, STAR 내용으로 검색

> **표시 정보**
> 
- 에피소드 카드 : 카테고리 배지, 경험명, 에피소드명 표시
- 기간 정보 : 경험 시작일 및 종료일 표시 (없으면 "-" 또는 "진행중")
- STAR 항목 : Situation, Task, Action, Result 각 항목별 내용 표시
- 강점/역량 : 에피소드에 태그된 역량 키워드 표시
- 작업 버튼 : 편집 가능한 에피소드에만 편집 버튼 표시

> **편집 기능**
> 
- 3단 레이아웃 편집 뷰
    - 왼쪽: 에피소드 목록 (프로젝트별 그룹화, 클릭하여 선택)
    - 중앙: STAR 입력 필드 (Situation, Task, Action, Result)
    - 오른쪽: 강점/역량 태그 관리 및 저장/취소 버튼
- STAR 입력
    - 각 항목별 텍스트 영역으로 자유롭게 입력
    - 저장 버튼 클릭 시 데이터베이스에 저장
- 강점/역량 태그 관리
    - 태그 추가 다이얼로그에서 역량 키워드 선택
    - 선택된 태그 표시 및 제거
    - 태그는 역량 키워드 목록에서 선택
- 기간 정보 편집
    - 달력 컴포넌트로 시작일/종료일 선택
    - 날짜 범위 선택 기능
    - "진행중" 표시 (종료일이 없는 경우)
- 저장 및 취소
    - 저장 버튼: STAR 내용, 태그, 기간 정보 모두 저장
    - 취소 버튼: 변경사항 취소 및 편집 모드 종료
